HKMOutputString 输出字符串

功能

用于向需要打字输入文本的地方输出字符串。设置输出字符串的模式使用HKMSetMode，设置输出速度使用HKMSetKeyInterval。如果输出的中文是乱码，则需要修改输出字符串的模式。要注意当前输入法，有的输入法不支持这样输入，会有中文无法输出或者全部无法输出的问题。

参数

  • 参数1: 设备对象：无类型指针。使用HKMOpen可以创建它。
  • 参数2: 字符串：字符串(Unicode/Ansi)。默认是Unicode字符串，可以通过HKMSetMode修改为Ansi字符串。

返回值

布尔值。成功返回True，失败返回False。

C#例子

uint dwDevId;
IntPtr lpDev;
dwDevId = SkmCore.HKMSearchDevice(0x1234, 0xABCD, 0);
if (dwDevId == 0xFFFFFFFF)
{
    Console.WriteLine("未找到键鼠模拟器");
    return;
}
lpDev = SkmCore.HKMOpen(dwDevId, 0);
if (lpDev == IntPtr.Zero)
{
    Console.WriteLine("打开键鼠模拟器失败");
    return;
}
SkmCore.HKMOutputString(lpDev, "输出字符串测试");
SkmCore.HKMClose(lpDev);
