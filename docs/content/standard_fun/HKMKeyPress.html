HKMKeyPress 键盘按键

功能

键盘的键按下再弹起。设置键盘按下和弹起时间间隔使用HKMSetKeyInterval。

参数

  • 参数1: 设备对象：无类型指针。使用HKMOpen可以创建它。
  • 参数2: 键名：字符串(Unicode/Ansi)或者无符号32/64位整型数(32位模块中是32位整型数，64位模块中是64位整型数)。按键名使用字符串(默认是Unicode字符串，可以通过HKMSetMode修改为Ansi字符串)，按键码使用无符号32/64位整型数。按键码和按键名可查看虚拟键码表。字符串支持组合键，组合键的按键名用"+"连接，例如："Ctrl+C"。

返回值

布尔值。成功返回True，失败返回False。

C#例子

uint dwDevId;
IntPtr lpDev;
dwDevId = SkmCore.HKMSearchDevice(0x1234, 0xABCD, 0);
if (dwDevId == 0xFFFFFFFF)
{
    Console.WriteLine("未找到键鼠模拟器");
    return;
}
lpDev = SkmCore.HKMOpen(dwDevId, 0);
if (lpDev == IntPtr.Zero)
{
    Console.WriteLine("打开键鼠模拟器失败");
    return;
}
SkmCore.HKMKeyPress(lpDev, "F1");
SkmCore.HKMClose(lpDev);
