HKMSetLightMode 设置指示灯模式

功能

设置键鼠模拟器的指示灯模式。

参数

  • 参数1: 设备对象：无类型指针。使用HKMOpen可以创建它。
  • 参数2: 模式：无符号32位整型数。可取值如下：
    • 0：闪烁(默认模式)
    • 1：常亮
    • 2：常灭
    • 8：模式不复位，盒子复位时模式不变，可与其它值做位或运算组合使用
  • 参数3: 是否鼠标：布尔值。取FALSE时，自动设置指示灯模式，如果同时有键盘模式或者鼠标模式的设备，设置键盘模式的设备的指示灯模式。取TRUE时，自动设置指示灯模式，如果同时有键盘模式或者鼠标模式的设备，设置鼠标模式的设备的指示灯模式。

返回值

布尔值。成功返回True，失败返回False。

C#例子

uint dwDevId;
IntPtr lpDev;
dwDevId = SkmCore.HKMSearchDevice(0x1234, 0xABCD, 0);
if (dwDevId == 0xFFFFFFFF)
{
    Console.WriteLine("未找到键鼠模拟器");
    return;
}
lpDev = SkmCore.HKMOpen(dwDevId, 0);
if (lpDev == IntPtr.Zero)
{
    Console.WriteLine("打开键鼠模拟器失败");
    return;
}
SkmCore.HKMSetLightMode(lpDev, 1, false);
SkmCore.HKMClose(lpDev);
