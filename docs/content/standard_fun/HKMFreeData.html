HKMFreeData 释放数据

功能

释放数据，用于释放一些函数返回的数据所占用的资源。相关函数：HKMSearchDeviceAll、HKMCheckPressedKeys

参数

  • 参数1: 数据地址：无类型指针。使用HKMSearchDeviceAll、HKMCheckPressedKeys可以获得此参数。

返回值

布尔值。成功返回True，失败返回False

C#例子

IntPtr lpStr = SkmCore.HKMCheckPressedKeys(1);
// 执行操作
SkmCore.HKMFreeData(lpStr);

C语言例子

LPWSTR lpStr;
lpStr=HKMCheckPressedKeys(1);
//执行操作
HKMFreeData(lpStr);
