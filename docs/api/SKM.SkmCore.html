HKMGetDevInfo(nint, uint, bool)

获得键鼠模拟器的设备信息

public static extern uint HKMGetDevInfo(nint HKMData, uint Index, bool Mouse)

Parameters

HKMData nint

设备对象：无类型指针。使用HKMOpen或者HKMOpen2可以创建它。

Index uint

序号,可取值如下:

  • 1: 设备类型，最新版返回2
  • 2: 固件版本,返回值中，0-15位是修订号，16-23位是副版本号，24-31位是主版本号。
  • 3: 运行时间,返回值是从通电或者复位开始运行的毫秒数，断电和复位都会归零。系统休眠时不计时。这个时间精度不高，有误差。
  • 4: 通电时间。返回值是从通电开始运行的毫秒数，断电会归零，复位不会归零。系统休眠时不计时。这个时间精度不高，有误差。
  • 6: 复位次数。
  • 7: 运行状态。设备未连接返回0xFFFFFFFF，设备准备中返回0，正常工作状态返回1，编辑状态返回2，禁止操作状态返回3。
  • 8: USB设备接口的VID值
  • 9: USB设备接口的PID值
  • 10: USB设备接口的设备版本值
Mouse bool

是否鼠标：布尔值

  • True: 获得鼠标模式的设备信息
  • False: 获得键盘模式的设备信息

Returns

uint

无符号32位整型数,返回值由参数2决定。 没找到或者失败返回0xFFFFFFFF。

HKMOpen(uint, uint)

打开设备

public static extern nint HKMOpen(uint DeviceId, uint DpiMode)

Parameters

DeviceId uint

设备ID：无符号32位整型数。使用HKMSearchDevice或者HKMSearchDevice2可以获得它。

DpiMode uint

DPI模式：无符号32位整型数。用于设置鼠标移动(HKMMoveTo和HKMMoveR2)和获得鼠标坐标(HKMGetCursorPos和HKMGetCursorPos2)时使用的坐标的DPI模式,可取值如下：

  • 0 每个显示器DPI感知
  • 1 主显示器DPI感知
  • 2 键鼠模拟器DPI感知
  • 3 DPI不可
  • 4 当前上下文的DPI感知

Returns

nint

设备对象：无类型指针。创建的设备句柄。失败返回NULL。不再使用它时用HKMClose可以关闭它以释放资源。
